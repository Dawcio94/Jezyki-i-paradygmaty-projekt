# Jezyki-i-paradygmaty-projekt
Zamieszczone są dwie solucje związane z dwoma różnymi paradygmatami programowania. Projekt związany z rozwiązywaniem układów równań metodą Gaussa jest zbudowany imperatywno-strukturalnie, zaś zagadnienie o 4 metodach wyznaczania miejsca zerowego funkcji wraz z podaną ilością kroków oraz czasem obliczenia x0 występuję w formie obiektowej.
